## Using 
1) Clone repository 
2) Install necessary libraries with reqrements.txt
3) In work_files craete directory data_base 
4) In work_files craete file .env with your vk api key. Example: API_KEY = "your_key" 
5) In work_files run main.py
6) Use local host
